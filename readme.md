# TechConf 2026 - Event Landing Page

## Description
Website for a tech conference, created for the Web Design course. Contains 4 HTML pages and external CSS.

## Scenario
**#10 - Tech Event Landing Page + Archive**

## Structure
- `index.html` - Homepage
- `about.html` - Conference details and speakers
- `archive.html` - Past events archive (table)
- `register.html` - Registration form
- `styles.css` - CSS styles

## Requirements Implemented
- Semantic structure (header, nav, main, footer)
- One h1 per page
- Consistent navigation across pages
- Skip link for accessibility
- Table with thead, tbody, tfoot and 6 rows
- Form with 10+ controls and labels
- Fieldset, legend, required
- External CSS and responsive design
- HTML and CSS validation passed
